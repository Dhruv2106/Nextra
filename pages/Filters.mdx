# Filters

The top line of the trace shows the filter control panel, that is (almost) the same for all different view of the tool. The first line has 5 input boxes: Event, Gateway, Device, Command and Message. These are input for basic text search filters, where each of the inputs correspond to a column in the event view table. A text in any of these field will filter the data to show only log records that match all the entered criteria.
